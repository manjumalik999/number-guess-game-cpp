# number-guess-game-cpp
🎮 A simple C++ Number Guessing Game where the user tries to guess the correct number. This beginner-friendly project demonstrates user input, conditional statements (if-else), variables, and console output, making it a great exercise for learning the fundamentals of C++ programming. 🚀

📖 README.md
🎮 C++ Number Guessing Game

A beginner-friendly C++ program where the user guesses a secret number between 1 and 10.

✨ Features
🎯 Guess the secret number
⌨️ Takes user input
✅ Checks if the guess is correct
💬 Displays appropriate messages
📚 Easy to understand for beginners

🛠️ Concepts Used
Variables
User Input (cin)
Output (cout)
if-else Statements

📌 Sample Output

🎮 Number Guessing Game
Guess a number between 1 and 10: 7

🎉 Congratulations! You guessed the correct number.

🚀 Learning Outcomes
Learn how to take user input.
Understand conditional statements.
Practice writing simple interactive C++ programs.
Build confidence with beginner-level programming concepts.
